Privacy Policy for eBay Inventory Dashboard
Last Updated: March 31, 2026

1. Overview
This application is a proprietary, private tool used by AMTcapital for internal inventory management and operational efficiency. It is not a public-facing service.

2. Data Collection & Access
We interact with eBay account data exclusively through the official eBay API. This includes:

Listing Data: Active listings, Item IDs, and category information.
Pricing Data: Current MSRP, floor pricing, and promotional status.
Inventory Status: Quantity on hand and SKU mapping.
3. Use of Data
The data collected is used solely for the following business purposes:

Visualizing inventory health via a private dashboard.
Automating synchronization between eBay and secondary marketing channels.
Internal financial reporting and revenue planning.
Note: We do not sell, rent, or share this data with third parties or use it for external marketing.

4. Storage & Security
Location: Data is stored in private JSON/database files on secured servers or within private GitHub repositories.
Retention: Data is kept only as long as necessary for active inventory management.
Credentials: All API keys and OAuth tokens are stored as GitHub Secrets and are never hard-coded or made public.
5. User Rights & Data Deletion
As the primary user, you may revoke the application's access to your eBay data at any time via your eBay Account Settings (under "Applications"). Upon request or decommissioning of the tool, all locally stored cache and JSON data will be permanently deleted.
